-------------------------------
Units
-------------------------------

Unit symbols used in the ``Unit`` column of the :ref:`group reference <group-reference>`.
A unit of ``–`` denotes a unitless (dimensionless) quantity.

.. list-table::
   :header-rows: 1
   :widths: 20 80

   * - Symbol
     - Meaning
   * - ``m/s``
     - meters per second
   * - ``m``
     - meters
   * - ``Hz``
     - Hertz
   * - ``s``
     - seconds
   * - ``V``
     - volts
   * - ``–``
     - unitless
   * - ``rad``
     - radians
   * - ``dB``
     - decibels
   * - ``#``
     - count
   * - ``%``
     - percent
   * - ``kg/m²``
     - kilograms per square meter
